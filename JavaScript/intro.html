Introduction to JavaScript

Javascript ― это интерпретируемый язык программирования общего назначения с динамической типизацией данных.

Интерпретируемый значит, что программа выполняется в исходном виде, т.е., исходный код не преобразыется в машинный или промежуточный код. В связи с этим ошибки на этапе компиляции и компоновки отсутствуют, есть только ошибки на этапе выполнения. Интерпретатор JavaScript может быть установлен в любую ОС, как семейство Windows, так и семейство Unix.
Это даёт хорошую переносимость кода или кроссплатформенность. JavaScript также иногда используют для разработки мобильных приложений под Android.
JavaScript также часто используют в тестировании для написания скриптов. Но несмотря на всё это, чаще всего JavaScript используется для разработки Web приложений, а именно FrontEnd части приложения. Основным и самым распространённым интерпретатором JavaScript является самый обычный браузер.

Динамическия типизация говорит о том, что тип переменных и констант определяется во время присваивания, по типу значения справа (r-value). Это в свою очередь означает, то что у переменной нет как такового постоянного типа данных. Например, в переменную, в которой хранится число можно записать строку. Это достигается благодаря тому, что абсолютно все типы являются объектами, а также благодаря полиморфизму.

Переменные и типы данных

Переменные и константы

var, let и const var объявляет глобальныую переменную, несмотря на область видимости, в которой она объявлена. Если записать значение в несуществующую переменную, то это не вызовет ошибку, это будет аналогично ключевому слову 'var'; let объявляет локальную переменную только для той области видимости, в которой она используется; const объявляет локальную константу. Константы могут быть только локальными;
